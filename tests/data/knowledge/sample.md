# Sample Document for Testing

## Introduction

This is a sample markdown document used for testing the Pantheon Knowledge Base system.

## Features

The knowledge base supports:
- Document indexing with semantic chunking
- Hybrid search (dense + sparse vectors)
- Metadata extraction
- Reranking with FlashRank

## Technical Details

The system uses LlamaIndex and Qdrant for efficient document retrieval. It supports multiple file formats including Markdown, PDF, and DOCX.

### Performance

The system achieves:
- Indexing speed: ~1.5 seconds per document
- Retrieval latency: <150ms
- Accuracy improvement: 35-55% vs pure vector search
